* Package Manager
:PROPERTIES:
:CUSTOM_ID: h1-package-manager
:END:

Packages are managed by [[https://github.com/radian-software/straight.el][straight.el]] with [[https://github.com/jwiegley/use-package][use-package]] integration.
All packages are installed from source via straight and configured
declaratively with =use-package=.

* Completion Framework
:PROPERTIES:
:CUSTOM_ID: h1-completion-framework
:END:

A Vertico-based completion stack:

- [[https://github.com/minad/vertico][Vertico]] -- vertical minibuffer completion UI
- [[https://github.com/oantolin/orderless][Orderless]] -- flexible matching (regexp, literal, initialism)
- [[https://github.com/minad/marginalia][Marginalia]] -- rich annotations in the minibuffer
- [[https://github.com/minad/consult][Consult]] -- enhanced search and navigation (ripgrep, line, buffer, etc.)
- [[https://github.com/oantolin/embark][Embark]] -- context actions on completion candidates

* Org Mode
:PROPERTIES:
:CUSTOM_ID: h1-org-mode
:END:

Org configuration lives in a separate file (=lisp/org-conf.el=) to
avoid version conflicts with straight.el.

- Agenda, habit tracking, time clocking
- Capture templates for quick notes and tasks
- Export to LaTeX, Beamer, HTML
- Clipboard image paste: =me/org-yank= detects images on the clipboard
  (raw data or file URIs) and saves them into an assets directory with
  auto-incrementing names

* Email
:PROPERTIES:
:CUSTOM_ID: h1-email
:END:

Mail configuration is in =lisp/mail-conf.el= (not public).  Contact
management via [[https://savannah.nongnu.org/projects/bbdb/][BBDB]].

:CUSTOM_ID: h1-editing
:END:

- [[https://github.com/Fuco1/smartparens][Smartparens]] (~C-c s~) -- structural editing
- [[https://github.com/magnars/multiple-cursors.el][Multiple cursors]] (~C-c m~) -- multi-cursor editing
